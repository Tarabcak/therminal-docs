Download complete yearly candle data as Parquet files. Each file contains all 1-minute OHLCV candles for all markets in that year.

## Request

<span class="http-method">GET</span> `/api/v1/download/candles`

### Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `year` | integer | **Yes** | Year (e.g., `2025`) |

## Example

```bash
curl "https://api.mostlyright.xyz/api/v1/download/candles?year=2025"
```

```python
from therminal import TherminalClient
client = TherminalClient()
df = client.download_parquet("candles", year=2025, load=True)
```

## Response

```json
{
  "url": "https://r2.cloudflare.com/...?signature=...",
  "key": "candles/2025.parquet",
  "expires_in": "5m"
}
```

### Parquet schema

| Column | Type | Description |
|--------|------|-------------|
| `market_ticker` | string | Market ticker |
| `period_interval` | int16 | Interval (minutes) |
| `end_period_ts` | timestamp | Candle end time (UTC) |
| `open` | int16 | Open price (cents) |
| `high` | int16 | High price (cents) |
| `low` | int16 | Low price (cents) |
| `close` | int16 | Close price (cents) |
| `volume` | int32 | Volume |
| `open_interest` | int32 | Open interest |
