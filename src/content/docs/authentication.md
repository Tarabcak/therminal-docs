---
title: Authentication
description: API authentication (coming soon)
---

:::note
The Therminal API is currently **public and read-only**. No API key is required.
:::

Authentication via Supabase Auth (JWT) is planned for a future release. When enabled:

- Bearer token in the `Authorization` header
- API keys for programmatic access
- Per-user rate limits

The current rate limit of 100 requests/second per IP applies to all callers.
